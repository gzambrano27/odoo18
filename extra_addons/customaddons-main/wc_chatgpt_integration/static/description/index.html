Overview

Enhance your team communication in Odoo by integrating ChatGPT directly into the Discuss app. With this module, users can interact with ChatGPT inside their chat channels or private messages, making collaboration smarter and faster.

Features

AI Assistant in Discuss :

ChatGPT is available as a virtual participant in your Odoo chats.

Instant Replies:

Ask questions and get quick, contextual answers within Discuss.

Secure & Configurable

Admins can control API keys, usage limits, and access rights.

Summarization

Summarize long chat conversations into clear, concise notes.

How to Create an API Key in OpenAI

Step 1: Log in to OpenAI :

1) Go to https://platform.openai.com/.

2) Sign in with your OpenAI account (Google, Microsoft, or email login).

Step 2: Navigate to API Keys :

1) After login, click your profile icon (top-right corner).

2) Select “View API keys” from the dropdown (link: https://platform.openai.com/account/api-keys).

Step 3: Create a New API Key :

1)Click the “+ Create new secret key” button.

2) A popup will appear with your new API key (a long string like sk-xxxxxxxxxxxxxxxx).
